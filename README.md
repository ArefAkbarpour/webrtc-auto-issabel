# ☎️ WebRTC Auto Setup for Issabel / Asterisk

**One-command WebRTC + SSL deployment** with built-in SIP IP restriction and full security — designed for DevOps engineers, VoIP providers, and network admins who want a clean, automated, and secure WebRTC setup in minutes.

---

## 🚀 Highlights

- 🔒 Auto-secured SIP registration (only 1 trusted host IP)
- 🧠 One-command installation — no manual config needed
- 🪄 Automatic Let’s Encrypt SSL for WebRTC (WSS)
- 🧱 Works on Issabel & pure Asterisk (PJSIP)
- 🔥 Includes ready-to-use HTML + JsSIP WebRTC client
- 🧰 Sets up firewall rules automatically
- 🌍 Production-ready in under 2 minutes
